## 📖 关于项目
本项目自动从不同来源收集和分类VPN配置（如V2Ray、Trojan和Shadowsocks等不同协议）。我们的目标是为用户提供更新和可靠的配置。

> **注意：** 对于过长或包含不必要字符（如大量`%25`）的配置，为确保质量，将进行过滤。
